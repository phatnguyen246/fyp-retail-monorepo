# Catalog Module Docs

Tài liệu của `catalog` module được colocate cùng code để phản ánh ownership rõ ràng hơn.

## Purpose

`catalog` sở hữu:

- storefront product listing/search/detail/compare
- admin catalog CRUD
- variant image management
- import/clone product
- technical mapping và query/index notes riêng của catalog
